drkSimon v1.5
------------------------------------------------------------------------

Introduction
------------
drkSimon est un jeu de mémoire pour smartphone. Il reprend le principe 
du célèbre jeu électronique des années 80′s simon.

Ce jeu est développé et maintenu par Philippe Bousquet 
<darken33@free.fr> depuis octobre 2013 et est distribué selon les 
termes de la GNU General Public License v3.

Les graphismes ont été réalisé par Philippe Bousquet et sont 
distribuées sous licence Creative Commons BY-SA.

Les effets sonores ont quant à eux été réalisé au travers de logiciels 
libres tels que Jackd, Qsynth et Audacity, et sont également 
distribuées sous licence Creative Commons BY-SA.

Comment jouer ?
---------------
Pour faire apparaître le menu il suffit d’appuyer sur la touche 
« menu », ou simplement effectuer un appui long sur l’écran, vous 
pourrez définir le niveau de jeu dans les paramètres.

Pour démarrer une partie, appuyez sur le « simon » il faut alors 
répéter les séquences de couleurs proposées par le jeu. Tant que vous 
ne faites pas d’erreur, la séquence se voit allongée d’une couleur.

Combien de temps allez vous tenir avant de vous faire avoir par simon ?
Bonne partie…

Technologies
------------
drkSimon est une application hybride dévelopée avec les technologies :
  - html5
  - PhoneGap
  - JQueryMobile
Au jour d'aujourd'hui elle a  étée packagée  pour le système android, 
mais toute personne qui le désire peut la packager pour les smartphones
suivant : iphone, windowsphone, blackberry, bada, ect...

Logiciel Libre
--------------
drkSimon est un logiciel libre, et a ce titre, en tant qu'utilisateur,
vous béneficiez des quatre libertés fondamentales :
  - Liberté 0 : La liberté d’exécuter le programme, pour tous les 
                usages.
  - Liberté 1 : La liberté d’étudier le fonctionnement du programme.
  - Liberté 2 : La liberté de redistribuer des copies.
  - Liberté 3 : La liberté d’améliorer le programme et de publier ses 
                améliorations.
